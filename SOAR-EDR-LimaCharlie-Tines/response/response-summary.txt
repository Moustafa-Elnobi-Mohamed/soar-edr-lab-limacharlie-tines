# Response Workflow Summary

This SOAR workflow performs the following response actions:

1. Receives detection events from LimaCharlie
2. Enriches and normalizes detection data in Tines
3. Sends alerts to Slack and Email
4. Prompts the analyst for isolation approval
5. Executes endpoint isolation upon approval
6. Confirms isolation status and notifies Slack

The workflow reduces mean time to respond (MTTR)
while maintaining analyst oversight.
