# MITRE ATT&CK Mapping

This detection maps to the following MITRE ATT&CK techniques
based on observed endpoint behavior.

## Credential Access

- **T1555 – Credentials from Password Stores**
  LaZagne targets stored credentials from browsers and system vaults.

- **T1003 – OS Credential Dumping**
  The tool attempts to extract credentials from operating system components.

## Execution

- **T1204 – User Execution**
  The tool is manually executed on the endpoint.

---

This mapping is used for classification and reporting purposes
and does not serve as the detection source itself.
